功能介绍：

  • 主题预览：
    在主题列表页面，每个主题新增了一个”预览“按钮，点击可预览主题，若无主题则预览标题详情。
  • 只看楼主：
    在主题页面，主题功能栏新增了一个“只看楼主”按钮，点击只看本页楼主的回复。
  • 突出楼主：
    回复列表中，楼主的回复自定义背景颜色(截图颜色为R:166 G:223 B:242 A:0.3)。
  • 突出感谢：
    回复列表中，心形感谢图标自定义颜色。
  • 会话详情/所有回复：
    在主题页面，回复新增了一个“会话详情”/“所有回复”按钮，点击查看本页内此用户与其回复内@的用户的会话详情/所有回复。
  • 用户信息：
    将鼠标悬浮在用户头像上半秒弹出该用户的主要信息。
  • 快速查看最近一条回复：
    将鼠标悬浮于回复内的“@某用户”可快速显示该用户本页内最近一条回复。
  • 插入表情：
    在回复文本框上，新增了一个“表情”按钮，点击可选择表情。
  • 插入图片：
    在回复文本框上，新增了一个“插入图片”按钮，点击可选择插入的图片文件，您也可以直接粘贴截图或拖拽图片文件至回复框直接上传。
  • 新建主题获取图片链接：
    在新建主题页面内，左侧添加一个弹出式图片管理窗口，可以粘贴截图/上传图片获取图片链接。
  • 消息提醒：
    在浏览器右上角有一个可以隐藏的图标，每5分钟检查一次新消息，当收到新消息时 Chrome 的通知中心会提示您，同时扩展按钮显示如下：
  • 一键签到：
    当您今日的登录奖励未领取时，您将不必来回跳转领取登录奖励，一键且优雅。
  • 自动签到：
    您可以在本页内开启自动签到功能，领取成功后您将获得来自 Chrome 通知中心的通知。
  • 旋转图片：
    当鼠标悬浮于回复内的图片时，右上角添加了两个功能键用以旋转一些不方便欣赏的图片。
  • 折叠超长主题、回复：
    开启后长度超过 1800px 的主题、长度超过 1000px 的回复将被折叠，需手动展开。
  • 新标签页浏览主题：
    开启后所有主题将在新标签页内打开。
  • 跳过主题：
    当您在同一主题内翻页将跳过主题内容直接到回复列表。
  • 回到顶部：
    论坛内双击空白处(非按钮、连接)即可返回顶部。

Q & A：

  • 如何修改打开消息界面的快捷键？
    扩展第一次安装默认快捷键为 Ctrl/CMD + E 如果需要修改成其他快捷键可以到浏览器的扩展管理页面 chrome://extensions/ 底部键盘快捷键处进行自定义。
  • MAC 下为什么复制图片文件粘贴至回复框都为文件图标？
    由于浏览器局限，直接粘贴文件不被支持，请剪切图片粘贴而不是文件，比如使用 QQ 的截取屏幕功能，或使用插入图片功能。
  • 插入图片支持什么格式？
    根据所使用图床而定，支持图床支持的所有格式，但推荐 png、jpg 格式。请勿上传图床不支持的文件，如 psd。
  • 使用表情功能需要登录微博吗？
    不需要。
  • 为什么一直提醒我有新消息？
    只要您没有查看新消息，v2ex plus 将会每5分钟提醒您一次。
  • 为什么我有时收不到 Chrome 通知中心的提醒？
    1. 请确保 Chrome 的通知中心设置中已经勾选 v2ex plus。
    2. 请确保系统通知中心不处于免打扰模式。
    3. Chrome for Mac 下，当屏幕正显示处于全屏状态的 Chrome 时，Chrome 将无法弹出通知，这是一个已知的待修复的BUG，此时您应该留意来自右上角图标的提示。
  • 我该如何选择图床？
    如果您从来没听说过 imgur，请选择微博图床，其他情况仍然推荐没有被特殊照顾的微博图床以保证图片显示速度。
  • 上传的图片是上传到哪里？
    图片上传至微博图床，需要登录对应账号但不会像“围脖是个好图床”一样以用户（@围脖是个好图床1 ……）发表图片的方式获得图片链接，所以在图片发送前，只有您和服务器知道图片内容，如果您使用微博图床扩展不会使用您的微博账号发布任何微博，上传至 imgur 则是通过 imgur 的 API 上传至在 imgur 注册的应用内。
  • 为什么我没有上传图片仍然提示有图片在上传？
    图片是在后台上传，即使关闭了上传图片的页面未上传完的图片仍在继续上传直到完成或出错。
  • 新建主题中想复制图片链接为什么一下选中一下不选中？
    您只需将鼠标悬浮于链接即自动全选，无需单击直接复制。
  • 为什么有时我查看了新消息，但图标任然提示有未读消息？
    如果您需要查看消息后图标立即更改，请点击图标或通知查看消息，否则图标最长需要5分钟才能有反应。
  • 我只需要消息通知的按钮提示，如何关闭不需要的横幅通知？
    横幅通知由系统或浏览器控制，您可以按需单独关闭，并不会影响按钮提示。
  • 什么是会话详情的智能模式？
    扩展程序会智能判断对话内的每条回复，不相关的将被隐藏，您也可以选择关闭此功能，每个会话详情默认打开。
  • 为什么很久都没自动签到？
    开启了自动签到后每小时触发一次，您不必特别留意，签到成功后 Chrome 通知中心会提醒您。每天（ UTC 时间）成功签到一个账号后，扩展将停止每小时的签到检测，切换账号请手动签到或关闭自动签到后再打开。
  • 自动签到和一键签到冲突吗？
    不冲突。
  • 为什么偶尔无法显示旋转图片的按钮？
    为了美观屏蔽宽度小于 100px 或高度小于 30px 的图片的旋转功能。
  • 什么是定时激活微博？
    为改善部分使用新浪微博为图床但不常登陆新浪微博的用户体验，故推出此实验性功能，主要用途是每小时访问一次 weibo.com 使用保存的 cookie 保持微博账号的登陆状态，此功能的稳定性及可靠性待确定。
    注意：使用 imgur 图床的用户无需此功能。
